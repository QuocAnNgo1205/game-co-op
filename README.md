
✅ Các bước để người khác mở project Unity từ GitHub:
1. Clone repo về máy
git clone https://github.com/QuocAnNgo1205/game-co-op.git
2. Mở Unity Hub
Chọn "Add" project
Trỏ đến thư mục vừa clone
3. Unity sẽ tự động tái tạo các file tạm
Unity sẽ tạo lại thư mục Library/, Temp/, v.v. khi mở project.
Không cần push các thư mục này lên GitHub.
